Contenu de la documentation de KhatovarWeb
******************************************

Bienvenue dans la documentation de KhatovarWeb, l’application
`Symfony <http://symfony.com/>`_ propulsant le site web de
`la compagnie franche du Khatovar <http://www.compagniefranchedukhatovar.fr/>`_.

Vous trouverez ci-dessous la table des matières renvoyant aux différentes
sections de l’application.

La documentation est encore en cours de construction et est complétée au fur et
à mesure de l’avancée des travaux sur l’application.

Il est indispensable de lire l’introduction avant de passer aux autres sections.
